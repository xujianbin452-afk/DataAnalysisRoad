# 数据说明

## 数据来源

本项目使用 Give Me Some Credit 公开数据。分析文件 `cs-training.csv` 下载自清华大学数据库研究组公开镜像：

`https://dbgroup.cs.tsinghua.edu.cn/datasets/finance/GiveMeSomeCredit/cs-training.csv`

原始文件保存在 `data/raw/`，不纳入 Git 版本控制。

## 字段说明

| 字段 | 业务含义 | 分析处理 |
|---|---|---|
| `timestamp` | 镜像附加的占位时间 | 全部为同一时间，删除 |
| `id` | 顺序记录编号 | 仅用于记录定位，不作为风险特征 |
| `SeriousDlqin2yrs` | 未来两年是否发生严重逾期或财务困境 | 目标变量 |
| `RevolvingUtilizationOfUnsecuredLines` | 无担保循环信贷额度使用率 | 检查大于1及极端值 |
| `age` | 年龄 | 检查0值及分组稳定性 |
| `NumberOfTime30-59DaysPastDueNotWorse` | 历史30至59天逾期次数 | 检查96、98等特殊值 |
| `DebtRatio` | 负债率 | 结合收入缺失检查极端值 |
| `MonthlyIncome` | 月收入 | 检查缺失客群风险 |
| `NumberOfOpenCreditLinesAndLoans` | 开放信贷账户及贷款数量 | 检查零值和极端值 |
| `NumberOfTimes90DaysLate` | 历史90天及以上逾期次数 | 重点风险变量，检查特殊值 |
| `NumberRealEstateLoansOrLines` | 房地产贷款或信贷账户数量 | 检查极端值和非线性关系 |
| `NumberOfTime60-89DaysPastDueNotWorse` | 历史60至89天逾期次数 | 重点风险变量，检查特殊值 |
| `NumberOfDependents` | 家属数量 | 检查缺失值及极端值 |

## 使用限制

数据缺少真实业务中的审批结果、贷款金额、定价、资金成本和完整还款时间，因此适合进行截面风险分析、客群分层和规则模拟，不适合直接计算真实风险收益或进行严格账龄分析。
